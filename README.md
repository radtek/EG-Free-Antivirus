# EG Free Antivirus
 
EG Free Antivirus is an open source antivirus software for Windows OS that uses [ClamAV](https://www.clamav.net) as an internal scanning engine for detecting trojans, viruses, malware & other malicious threats.

# Building Source code 

To build the source code, download [wxwidgets](https://www.wxwidgets.org/downloads/) version 3.0.2, build it (with VS 2008 for XP, with VS 2013 for above XP) and add environment variable named as (WX_DIR_XP for XP, WX_DIR_7 for above XP) for location (â€¦/wxWidgets-3.0.2). For XP Open the VS solution file in VS2008 and build the code, for above XP open the VS solution file in VS2013. The ClamAV executables can be build from [clamav with additional patches](https://github.com/clamwin/clamav) .
